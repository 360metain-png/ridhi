## About Ridhi

Ridhi is an **India-based, globally first social universal app** made by **Krilo Digitech Pvt Ltd**. It combines social networking, dating, short-form video, live streaming, audio rooms, podcasts, and creator monetization into one powerful platform.

Think **ShareChat × Instagram × TikTok × Spotify**, built for India and the world. Available on Android & iOS via Expo/React Native.

### Company
- **Company Name**: Krilo Digitech Pvt Ltd
- **Product**: Ridhi
- **App Type**: Social Universal App
- **Platforms**: Android & iOS
- **Target Market**: India (first), Global expansion
- **Founded**: 2026
- **Funding**: Seed Round, $2.5M Raise

### What Makes Ridhi a "Social Universal App"
Unlike competitors that focus on single verticals (Instagram for social, TikTok for short video, Tinder for dating, Spotify for audio), Ridhi integrates all these into one seamless ecosystem. This eliminates the need for Indian users to juggle 4+ apps.

## Advanced Technologies

### 1. Mobile Stack (Expo + React Native)
- **Expo SDK 54** with React Native
- **Expo Router** — File-based navigation with zero-config deep linking
- **React Compiler** — Babel plugin for performance optimization
- **Reanimated** — 60fps complex animations
- **Expo Blur + Linear Gradient** — Glassmorphism UI effects
- **TanStack Query** — Server state management with caching
- **Expo AV** — Video/audio playback for reels, podcasts, and audio rooms
- **Expo Image Manipulator** — Content processing and watermarking

### 2. Backend (Express 5 + PostgreSQL)
- **Express 5** — ESM-native Node.js server
- **PostgreSQL** — Relational database
- **Drizzle ORM** — Type-safe SQL builder
- **WebSockets (`ws`)** — Real-time chat, call matching, and live podcast rooms
- **Match Engine** — Server-authoritative cross-gender matching with per-second coin billing
- **Pino Logging** — Structured, high-performance logging
- **Rate Limiting** — API protection

### 3. AI / ML Features
- **AI Matchmaking Engine** — Weighted compatibility scoring
- **AI Content Moderation** — Local regex for Indian slang + server-side AI
- **AI Icebreaker Generator** — Multilingual (Hindi, Bengali, Telugu, etc.)
- **AI Reply Suggestions** — Smart conversation prompts
- **Conversation Quality Analysis** — Detects stale chats
- **Ridhi AI Bot** — Translation, captions, moderation, podcast show notes

### 4. Identity & Payments
- **SurePass** — Aadhaar/PAN/Bank e-verification for KYC
- **Razorpay, Cashfree, PhonePe, Instamojo** — Multi-provider payment architecture
- **Firebase Phone Auth + MSG91 SMS** — Dual OTP with auto-fallback

### 5. Admin & Operations
- **React + Tailwind + shadcn/ui** — Admin dashboard
- **Audit Trail System** — Every admin action logged permanently
- **Super Admin Panel** — Feature flags, VIP plan editor, coin spend rates
- **Geo-Targeting** — City-level and radius-based campaign targeting

### 6. Video & Content
