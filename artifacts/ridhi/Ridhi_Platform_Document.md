## 2. Income Sources & Revenue Streams

### Primary Revenue Streams (Platform)

#### 1. VIP Subscriptions — ₹12L MRR
- 4-tier recurring subscription model (Silver → Diamond Elite)
- Weekly, monthly, and yearly billing cycles
- Auto-renewal with prorated upgrade credits
- Revenue: ₹12 Lakhs/month (projected)

#### 2. Coin Economy — ₹8L/month
- In-app virtual currency for gifts, boosts, premium features
- Recharge packs: ₹49 to ₹4,999
- Coin acquisition: direct purchase, missions, daily rewards, ads, referrals
- Coin value: 1 coin = ₹1.00 (Super Admin configurable)
- Revenue: ₹8 Lakhs/month (projected)

#### 3. Live Stream Gifts — ₹5L/month
- 50+ virtual gift types (Rose 🌹 = 10 coins, Heart 💓 = 50, Ring 💍 = 500, Sports Car 🚗 = 5,000, Castle 🏰 = 50,000)
- Revenue split: 70% Creator / 20% Platform / 10% Agent
- Revenue: ₹5 Lakhs/month (projected)

#### 4. Ad Platform — ₹3L/month
- Inline banner ads injected every 5th post
- Self-serve campaign manager with targeting (city, age, gender, interests)
- Business ads, special client ads, commercial banners, promotions
- Revenue: ₹3 Lakhs/month (projected)

#### 5. Brand Marketplace — ₹2L/month
- Creator-brand deals, lead forms, post deals, brand registration
- 10% platform fee on all transactions
- Revenue: ₹2 Lakhs/month (projected)

#### 6. Creator Plans — ₹1L/month
- 3-tier Creator Pass subscriptions (Starter ₹199, Pro ₹499, Elite ₹999)
- Platform commission reduction: Free 30% → Starter 20% → Pro 13% → Elite 8%
- Revenue: ₹1 Lakh/month (projected)

#### 7. Shop Purchases — ₹1.5L/month
- In-app e-commerce with product variants, cart, checkout
- Platform margin on all shop transactions
- Revenue: ₹1.5 Lakhs/month (projected)

### Agent Income
- Commission: 2–10% of host's total gift earnings (based on agent level)
- Commission is paid separately by the platform — does NOT reduce host earnings
- Hosts are never left without management (auto-transfer to Admin oversight if agent leaves)

### Payout System
- Minimum withdrawal: ₹500
- Processing time: 7 working days
- Methods: Bank Transfer (NEFT/IMPS) or UPI ID
- Platform fee on withdrawal: 2%
- KYC mandatory before first withdrawal
- TDS applies on payouts above ₹10,000 as per Indian tax regulations

## 3. User Experience Flow

### Onboarding Flow (First Launch)
```
1. Splash Screen → Ridhi logo animation
2. Onboarding Carousel (3 slides)
   ┌──────────────────────────────────────────────────┐
   │ Slide 1: "India's Own Social Universe"                   │
   │ Slide 2: "Create, Connect, Earn"                        │
   │ Slide 3: "13 Languages. 1 Platform."                    │
   └──────────────────────────────────────────────────┘
3. Login Screen
   → Phone number or Email
   → 6-digit OTP (MSG91)
   → Fallback demo OTP for testing
4. Profile Setup (4 steps)
   Step 1: Full Name
   Step 2: Age & Gender
   Step 3: City
   Step 4: Interests (multi-select)
5. Home Feed → Welcome! You're in.
```

### Daily User Journey (Free User)
```
Morning ────────────────────────────────────────────────────────────────────
├── Open app → Stories bar at top (tap to view friends' stories)
├── Scroll Home Feed (For You / Trending / Local / Following / Community)
├── Like, comment, repost, share to WhatsApp Status
├── Watch Reels (vertical swipe, loop playback)
└── Claim Daily Coin Reward (Profile → Wallet)

Afternoon ────────────────────────────────────────────────────────────────────
├── Explore tab → Discover trending posts, suggested users, hashtags
├── Communities → Browse 10 categories, join interesting groups
├── Match tab → Swipe dating cards, Super Like someone special
└── Chat → Message matches, send emoji, share media

Evening ────────────────────────────────────────────────────────────────────
├── Join a Live Stream → Watch, comment, send virtual gifts
├── Audio Rooms → Listen to podcasts, Q&A sessions
├── Random Call → Match with a stranger (audio or video)
└── Create a Post → Text, photo, video, or poll
```

### Creator Journey (Monetization Path)
```
1. Create Content
   ├── Upload photos, videos, reels, voice reels
   ├── Create polls, stories, live streams
   └── Use AI Assistant for captions & hashtags (3 free/day)

2. Build Audience
   ├── Consistent posting → Followers grow
   ├── Story Analytics → Understand what content works
   ├── Collab Posts → Cross-pollinate audiences (1,000 coins)
   └── Pinned Posts → Showcase best work

3. Upgrade to Creator Pass
   ├── Starter (₹199/mo): Go Live, Podcast Studio, Basic Analytics
   ├── Pro (₹499/mo): HD Upload, Boost & Ads, Fan Club, Advanced Analytics
   └── Elite (₹999/mo): Brand Deals, Ridhi Radio, Account Manager, Revenue Projection

4. Go Live
   ├── Stream title + category selection
   ├── Receive virtual gifts from viewers
   ├── PK Battles with other hosts
   └── Gift earnings accumulate in coin wallet

5. Withdraw Earnings
   ├── Complete KYC (Aadhaar + PAN)
   ├── Request Payout (min ₹500)
   ├── Super Admin approves within 7 days
   └── Bank Transfer or UPI
```

### VIP User Journey
```
1. Subscribe to VIP (Profile → VIP Plans)
   → Silver (₹49/wk), Gold (₹99/wk), Platinum (₹199/wk), Diamond Elite (₹299/wk)

2. Unlock Premium Perks
   ├── Ad-free experience across all feeds
   ├── Unlimited polls (vs 1/week for free users)
   ├── Extra pinned posts, story highlights, saved posts
   ├── Group video calls at discounted rates
   ├── AI Content Assistant unlimited (Platinum+)
   ├── Shop discounts (10% Platinum, 15% Diamond)
   └── Bonus coins credited monthly (100–3,000)

3. Enhanced Social Presence
   ├── VIP badge on profile and in chat
   ├── Priority in live stream discovery
   ├── Priority in match suggestions
   └── Early access to new features
```

### Event Host Journey
```
1. Discover Events (Explore → Nearby Events)
   ├── Browse by category: Music, Food, Sports, Tech, Art, Social, Dating
   ├── Tabs: Nearby, Trending, My Events
   └── RSVP with one tap

2. Host an Event (500 coins)
   ├── Set title, description, date, location
   ├── Choose online or in-person
   └── Promote to followers

3. Manage Attendees
   ├── See RSVP count and attendee list
   ├── Send broadcast updates
   └── Post event photos to feed
```

### Settings & Privacy
```
Profile → Settings
├── Theme: Light / Dark / System
├── Language: English + 12 Indian languages (Hindi, Tamil, Telugu, Bengali, etc.)
├── Notifications: Push, Email, In-App toggles
├── Privacy: Activity Status, Profile Visibility, Blocked Users
├── Security: Change Password, 2FA, Login History
├── Account: KYC Status, Delete Account
└── Support: Help & Support, Report a Problem
```
